# 实物商品闲鱼文案模板

## 模板结构

用于二手商品、闲置物品、全新未拆封等实物商品的闲鱼文案。强调产品实际状态、交易保障和发货方式。

---

## 核心模块

### 第一部分：商品标题和亮点

```
【商品状态】商品名称 + 核心卖点（3-5个）
```

**示例：**
```
【95新】Apple AirPods Pro 2代 真无线降噪耳机 原装正品 保一年 听歌神器
【全新未拆封】索尼WH-1000XM5 头戴式降噪耳机 官方正品 全国联保 现货速发
【8成新】Nike Air Jordan 1 复古篮球鞋 US42码 实物拍摄 喜欢可小刀
```

---

### 第二部分：产品详情

```
📦【产品详情】
```

**必含内容：**

1. **产品基本信息**
   - 品牌/型号
   - 购买渠道和时间
   - 是否有发票/保修卡

2. **使用状况**
   - 成色评级（全新/99新/95新/9成新/8成新）
   - 使用时间
   - 使用频率

3. **外观状态**
   - 有无划痕、磕碰
   - 配件完整性
   - 包装情况

4. **功能测试**
   - 主要功能是否正常
   - 是否有故障
   - 是否维修过

**示例：**
```
📦【产品详情】
- 品牌：Apple AirPods Pro 2代
- 购买时间：2024年3月（苹果官网购入）
- 发票保修：有电子发票，保至2025年3月

💧使用状况：
- 成色：95新，几乎无使用痕迹
- 使用时间：约4个月
- 使用频率：每天1-2小时

🎨外观状态：
- 充电盒：无划痕，无磕碰
- 耳机本体：无污渍，干净整洁
- 配件齐全：原装充电线、说明书、替换耳塞完整
- 有原装包装盒

✅功能测试：
- 降噪功能：正常，效果很好
- 通话质量：清晰无杂音
- 连接速度：秒连iPhone
- 电池续航：耳机单次5小时，充电盒仍能充满3次
- 无维修史，无进水
```

---

### 第三部分：出闲置原因

```
💡【出闲置原因】
```

**常见原因分类：**

1. **设备升级** - "换了新款"
2. **闲置多时** - "买了很少用"
3. **搬家清理** - "搬家人带不走"
4. **工作变动** - "离职了不需要了"
5. **重复购买** - "朋友送了一个"

**示例：**
```
💡【出闲置原因】
换了iPhone 16，送的耳机用不上，所以出闲置。希望有缘人带走。
```

---

### 第四部分：交易保障

```
✅【交易保障】
```

**必含内容：**

1. **正品承诺**
2. **发货方式**（快递/同城自提）
3. **验货政策**
4. **售后说明**

**示例：**
```
✅【交易保障】
- 原装正品，支持验货
- 顺丰包邮（除偏远地区）
- 到货请当面验货确认收货
- 购买前可要求视频验货
- 签收后不支持退货
```

---

### 第五部分：发货信息

```
🚚【发货信息】
```

**示例：**
```
🚚【发货信息】
- 发货地区：北京市朝阳区
- 发货方式：顺丰速运
- 发货时间：下单后24小时内
- 运费：包邮（新疆/西藏/青海除外，邮费差价20元）
```

---

### 第六部分：购买须知

```
⚠️【购买须知】
```

**必含内容：**

1. **商品性质说明**（二手/全新/闲置）
2. **不可退货说明**
3. **验货提醒**
4. **售出不退原则

**示例：**
```
⚠️【购买须知】
- 本商品为二手闲置，非全新商品
- 二手商品售出后不支持退货换货
- 到货请及时验货，确认无误后再签收
- 谨防到手刀、恶意退货，爽约者拉黑
- 现货实拍，所见即所得
```

---

### 第七部分：关键词标签

```
🏷️关键词：[关键词列表]
```

**关键词策略：**
- 品牌词
- 产品类型词
- 型号词
- 特征词（全新/二手/闲置）
- 同义词

**示例：**
```
🏷️关键词：AirPods Pro 2 苹果耳机 降噪耳机 真无线 蓝牙耳机 二手耳机 闲置耳机 苹果配件 AirPods
```

## 特殊品类模板

### 电子产品（手机、电脑、耳机）

**补充说明：**
- 电池健康度（手机/平板）
- 屏幕状态（是否有亮点、坏点）
- 是否激活/锁机
- 系统版本
- 是否有ID锁/设备锁

### 服装鞋帽

**补充说明：**
- 尺码（身高体重对照）
- 是否有污渍
- 是否有异味
- 洗涤方式
- 材质说明

### 家电/数码配件

**补充说明：**
- 是否有保修
- 电源适配器情况
- 是否有配件齐全

---

## 排版要点

1. **使用emoji区分模块** - 📦🚚⚠️✅💡🏷️
2. **信息分层清晰** - 主要信息加粗，详细说明分段
3. **诚实描述** - 不隐瞒瑕疵，不夸大卖点
4. **图片引导** - 提示"实拍图在下方，请仔细查看"
5. **成交话术** - 结尾可加"喜欢的直接拍，爽快包邮"
