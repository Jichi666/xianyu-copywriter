# 闲鱼文案生成器

## 概述

帮助用户生成符合闲鱼平台特点的商品文案，包括产品介绍、使用说明、注意事项、版权声明等模块。基于成熟文案结构，降低售后纠纷风险，提升商品转化率。

## 核心能力

### 1. 虚拟商品文案生成

虚拟商品（软件、教程、模板、数字内容等）具有特殊风险和注意事项。生成时必须包含以下模块：

**必含模块：**
- ⚠️ **开头重点警告** - 明确限制条件（年龄、技术门槛等）
- 💰 **购买须知** - 无退款、虚拟商品不可退款、无人工售后
- 📦 **产品介绍** - 编号分段的产品描述
- 🔒 **版权声明** - 禁止二传二改、二次盈利
- 🏷️ **关键词标签** - 底部堆叠搜索关键词

**参见：** `references/virtual-product-template.md` 完整模板

**使用场景：**
- 用户说："帮我写一个Python课程的闲鱼文案"
- 用户说："生成一个AirPods教程的闲鱼商品描述"
- 用户说："我的软件产品需要闲鱼文案"

### 2. 实物商品文案生成

实物商品需要突出产品价值和交易保障：

**核心模块：**
- 🎯 **产品亮点** - 核心卖点3-5个
- 📸 **产品详情** - 规格参数、使用情况
- ✅ **交易保障** - 正品保证、发货时效
- ⚠️ **注意事项** - 不适用人群、售后政策

**参见：** `references/physical-product-template.md` 完整模板

**使用场景：**
- 用户说："帮我写二手iPhone的闲鱼文案"
- 用户说："生成一个运动鞋的闲鱼商品描述"
- 用户说："我的闲置物品需要合适的文案"

### 3. 文案优化

基于现有文案进行结构化优化：

**优化要点：**
- 添加明确的注意事项和风险提示
- 优化排版（使用emoji分隔）
- 补充版权声明（针对虚拟商品）
- 生成关键词标签

**使用场景：**
- 用户说："帮我优化这个文案：[现有文案]"
- 用户说："这个文案缺少什么？需要补充什么"

### 4. 关键词生成

根据产品特点生成高频搜索词：

**生成策略：**
- 核心产品词 + 品牌词
- 相关品类词
- 同义词、替代词
- 使用场景词

**使用场景：**
- 用户说："为这个产品生成搜索关键词"
- 用户说："帮我优化文案的关键词"

## 工作流程

### 步骤1：信息收集

从用户获取以下信息：
- **商品类型**：虚拟商品还是实物商品
- **产品名称**：核心产品名称
- **核心卖点**：3-5个主要特点
- **目标用户**：适合人群/限制条件
- **售价范围**：可选，影响预期层次
- **特殊说明**：版权、授权、售后政策等

### 步骤2：模板选择

根据商品类型选择对应模板：
- 虚拟商品 → `references/virtual-product-template.md`
- 实物商品 → `references/physical-product-template.md`

### 步骤3：内容填充

按照模板结构，将用户提供的信息填充到各模块：
- 使用用户提供的原始信息
- 补充必要的注意事项（虚拟商品必须有）
- 生成合适的关键词标签

### 步骤4：优化润色

- 确保排版清晰（使用emoji、分段）
- 检查必含模块是否齐全
- 确保语言风格符合闲鱼平台习惯
- 虚拟商品强调"无法退款"

### 步骤5：输出文案

直接输出完整文案，用户可直接复制使用

## 最佳实践

### 文案风格

- **简洁明了** - 避免冗余，直击要点
- **风险前置** - 虚拟商品开头即说明限制条件
- **结构清晰** - 使用emoji和编号分段
- **诚实透明** - 不夸大，不隐瞒缺陷

### 降低纠纷技巧

1. **明确告知** - 虚拟商品强调"无法退款"、"无人工售后"
2. **技术门槛** - 说明使用难度（"需要一定操作能力"）
3. **版权保护** - 根据需要设置版权声明（禁止二传二改、二次盈利）
4. **预期管理** - 说明产品能力边界（"能力有限，避免完美主义预期"）

### 排版规范

使用emoji增强可读性：
- ⚠️ 警告/注意事项
- 💰 价格/退款政策
- 📦 产品介绍
- 🔒 版权声明
- ✅ 交易保障
- 🏷️ 关键词标签

## 示例对话

**Q: 帮我写一个闲鱼文案，卖Python编程视频教程**

A: 需要以下信息：
1. 教程内容包含什么？（基础/进阶/实战项目）
2. 教程数量和时长？
3. 售价？
4. 有什么特点？（实战项目、代码可运行等）

提供信息后，我会生成包括以下模块的完整文案：
- 警告说明（不适合零基础等）
- 购买须知（虚拟商品无退款）
- 产品介绍（分编号详细说明）
- 版权声明（禁止二传二改）
- 搜索关键词

**Q: 优化这个文案：[用户粘贴文案]**

A: 我会检查：
1. 是否有明确的风险提示？
2. 是否有版权声明？
3. 排版是否清晰？
4. 关键词是否足够？

然后给出优化后的完整版本。
